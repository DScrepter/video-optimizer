Оптимизация видео

Простая утилита оптимизации видео для вставки на сайт в качестве фона

Поддерживаемые форматы: mp4, mov, avi, mkv, webm.

После обработки в папке с исходным видео появятся:

  • WebM и MP4 версии для десктопа (720p по умолчанию)
  • WebM и MP4 версии для мобильных устройств (480p по умолчанию)
  • Постер (poster.webp)
  • HTML-файл с тегом video для вставки на сайт
